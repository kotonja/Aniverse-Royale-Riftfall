# Aniverse Royale: Riftfall Roadmap

## AR-001 Foundation + Combat Sandbox - Complete

Docs, Rojo-friendly source structure, shared config, server-authoritative combat, client movement/input/HUD, two prototype Riftborn, greybox Training Dojo, damage dummies, and placeholder future services.

## AR-002 Studio Foundation + Movement / Camera / Game Feel Polish - Complete

Real Studio instance contract for the Training Dojo and HUD, dedicated server movement validation, remote rate limiting, sprint/dash/air dash/double jump/slide/wall kick feel, third-person camera pulses, mobile buttons, combat feedback, and temporary client VFX.

## AR-003 Four Character 3v3 Arena

Add two more original Riftborn and build a tighter 3v3 arena test loop.

## AR-004 Down/Revive + Aura Link

Add downed state, revive flow, squad pings, and first Aura Link combo prototypes.

## AR-005 Mini Royale 12-18 Players

Prototype a smaller royale loop with drop/spawn flow, eliminations, simple loot, and match end.

## AR-006 Lobby + Global Matchmaking

Build Rift Hub, MemoryStore matchmaking queues, parties, and reserved server teleport flow.

## AR-007 Battle Royale Map V1 + Riftstorm

Build Riftfall Archipelago V1, storm phases, safe zones, traversal, and map landmarks.

## AR-008 Progression + Data + Leaderboards

Add data profiles, wins, eliminations, seasonal stats, and safe leaderboard foundations.

## AR-009 Ranked + Cosmetics + Monetization

Add ranked and cosmetics only after the foundation is fun and stable. Monetization must comply with Roblox policy and avoid pay-to-win.

## AR-010 Public Beta

Stabilize onboarding, performance, matchmaking, balance, and release readiness.
